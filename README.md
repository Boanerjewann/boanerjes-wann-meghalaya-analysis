# Constituency Data Analysis

## Overview
This project analyses constituency-level development data in Meghalaya using Excel. The analysis focuses on infrastructure, scheme delivery, and fund utilisation patterns.

---

## Tools Used
- Microsoft Excel (data analysis, visualisations, dashboard)

---

## Sections

### Section 1: Data Audit
- Summary statistics created
- Data quality issues identified
- Derived columns:
  - Infrastructure Index
  - Welfare Dependency

### Section 2: Analysis
- Regional disparities analysed
- Scheme delivery gaps identified
- Fund utilisation patterns explored
- Key anomaly identified

### Section 3
- Not implemented in Python
- Analysis was completed in Excel instead

### Section 4: Dashboard
- Interactive dashboard created using slicers
- Filters: Region, District
- Indicators:
  - JJM coverage
  - Literacy rate
  - Fund utilisation
- Top and Bottom performers highlighted

---

## How to Use

1. Open the Excel file in `notebooks/analysis.xlsx`
2. Navigate through sheets:
   - Summary stats
   - Q1–Q4 analysis
   - Dashboard

---

## Assumptions
- Infrastructure Index equally weights roads, water, and connectivity
- Welfare Dependency uses BPL % and MGNREGS intensity
- Regional averages represent broad development patterns

---

## Limitations
- No automation using Python
- Static analysis (Excel-based)
- Limited to available indicators
